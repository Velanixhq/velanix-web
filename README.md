# Velanix

Velanix is a global fan network connecting passionate communities across the world.

## Vision
To build a safe, scalable and modern platform where fans can connect, share and grow together.

## Roadmap (V1)
- Landing page
- User authentication
- Public community feed
- Friend system
- Messaging (basic)
- Fan hub section

## Tech Stack (Planned)
- Next.js
- Tailwind CSS
- Node.js
- MongoDB
- Vercel

---

Velanix is an independent platform and is not affiliated with any official organizations.
